# Lumber Simulation

Это небольшая веб‑симуляция производственной цепочки, где три работника последовательно
обрабатывают ресурсы:

1. **Лесоруб** – рубит бесконечный лес и производит бревна.
2. **Пильщик** – превращает бревна в доски (1 бревно → 2 доски).
3. **Столяр** – из 4 досок делает забор.

## Структура проекта

```
├─ index.html            # Основная HTML‑страница
├─ src/
│  ├─ css/
│  │   └─ style.css      # Пользовательские стили (на основе Pico CSS)
│  └─ js/
│      ├─ utils.js       # Вспомогательные функции (обновление UI)
│      └─ game.js        # Основная логика игры
└─ package.json          # npm‑скрипты и зависимости
```

## Запуск

1. Установите зависимости:
   ```bash
   npm install
   ```
2. Запустите локальный сервер:
   ```bash
   npm start
   ```
   Откройте в браузере `http://127.0.0.1:8080` (по умолчанию).

## Тестирование

Просто откройте страницу в браузере и убедитесь, что индикаторы прогресса и счётчики
ресурсов обновляются в реальном времени. Если всё работает, значит рефакторинг прошёл
успешно.

## Плюсы рефакторинга

* Код разделён на логические модули – легче поддерживать и расширять.
* Стили вынесены в отдельный файл, можно добавлять свои правила без изменения HTML.
* Используется ES‑модули, что упрощает импорт/экспорт функций.
